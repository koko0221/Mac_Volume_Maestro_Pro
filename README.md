# 🎧 **Mac 音量調整器 Pro**

專為 Mac 直推有線耳機用戶設計，解決耳機被高音量「炸耳」的問題！以實用、美觀的 GUI 為目標，提供更佳音量調節體驗。

✨ **使用技術**：Python 3.13  
_💡 特別感謝 Bing AI 助力開發（非工商）_
