# 🎧 **Mac 音量調整器 Pro**

身為Mac直推的有線耳機用戶設計，因飽受突然被高音量炸耳的經驗(個人習慣從`音訊MIDI設定`調音量)，而起身力行使用AI製作出適合使用者的調音量GUI。

目前想省電省效能的人可以在要用的時候再開就好，不然他會一直運行，之後有機會我再做修正。

✨ **使用Python 3.13開發** 

_💡 Bing AI 協助開發_
